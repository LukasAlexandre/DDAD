# Feedbacks

> Feedbacks de fechamento dos blocos desta sessão, via `ddad feedback create --block <bloco> --session <sessao>`.

Um bloco sem feedback preenchido aqui não está concluído — está apenas implementado. O feedback é o que permite a um agente futuro entender o que de fato aconteceu sem reconstruir o raciocínio a partir do diff. Ver `Docs/00_ddad/metodologia.md`, seção 6.
