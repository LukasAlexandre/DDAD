# Prompts

> Prompts gerados a partir dos blocos desta sessão, via `ddae-engine prompt create --block <bloco> --session <sessao>`.

Cada prompt é autossuficiente: pode ser colado diretamente em qualquer agente de IA (Claude Code, Codex, Cursor, Copilot) sem depender de contexto de conversa anterior. Se um prompt depende de algo que não está escrito nele mesmo ou no bloco referenciado, ele está incompleto.
